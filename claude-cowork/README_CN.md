# Claude Cowork

> **[English Version](README.md)**

我怎么用 Claude 搞定实验报告、统计作业、长论文这些复杂作业。光粘贴题目扔给 ChatGPT 是做不好的，得教它你教授到底想要什么。

靠两个东西：**Skills** 和 **Chrome MCP**。

## Skills：让 Claude 知道你教授的脾气

Skill 就是你给 Claude 写的一份说明书，放在 `.claude/` 目录里。不是泛泛地说"写好一点"，而是针对你这门课具体校准过的。

你往里放三样东西：

1. **评分标准** — 原封不动抄进去，别概括
2. **教授的怪癖** — 课上说过但没写进标准的东西（"不要用被动语态"、"必须有百分误差"、"别画饼图"）
3. **历史扣分记录** — 每次被扣分就加一条，下次不再犯

两三次作业之后，Skill 就摸清你教授的口味了。Claude 不再写那种通用的 A- 作业，而是写出*拿红笔那个人*想看到的 A 作业。

## Chrome MCP：全流程自动化

这里是重点。Claude Cowork 通过 [Chrome MCP](https://browsermcp.io/) 连接你的浏览器，Claude 能直接看到网页、操作网页。

实际用起来就是：你把 Claude 指向作业页面，它能读题目要求、查资料、生成内容、填进文本框、在 tab 之间切换——全自动。在线实验报告表单、Canvas 作业提交这种，整个流程可以不用你动手。

我拿这个做那些原本要花几个小时在 Claude 和提交平台之间来回粘贴的长作业。Claude 从课程页面读评分标准，按 Skill 生成内容，直接填进去。

## 迭代循环

```
第一次作业：按评分标准写 Skill → 交 → 拿反馈
第二次作业：把反馈加进 Skill → 交 → 分数提高了
第三次作业：Skill 校准好了 → 稳定出好成绩
```

第一次可能不完美，没关系。每次反馈都让 Skill 更精准。到第三四次，你基本上只需要过一遍 Claude 写的东西就能交了。

## 适合什么作业

- 有严格格式要求的实验报告
- R/统计分析作业（代码和解读都要写好）
- 有评分标准的结构化论文（APA 引用、特定章节要求）
- 教授每次按同一套标准打分的重复性作业

不太适合创意写作或者教授想看你独特观点的作业。那些还是得你自己动脑。
